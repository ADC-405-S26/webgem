# A toy dataset for demonstrating the functions in webgem

A toy dataset for demonstrating the functions in webgem

## Usage

``` r
webgem_data
```

## Format

### `webgem_data`

A data frame with 25 rows and 12 columns:

- player:

  The name of the player being analyzed

- wrc_plus:

  Weighted runs created plus

- ops:

  Test score of each subject

- obp:

  Words recalled by each subject

- slg:

  Slugging percentage

- avg:

  Batting average

- iso:

  Isolated power

- war:

  Wins above replacement

- woba:

  Weighted on-base average

- bb_pct:

  Walk percentage

- barrel_pct:

  Barrel percentage

- hard_hit_pct:

  Hard-hit percentage
